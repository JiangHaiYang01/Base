
# BaseMvvM
Android 基础手脚架 


# Overview

简化开发 MVVM 框架 
最早之前其实写过一个base 的基础库框架。幻想着吧好多东西都集成进去，发现反而显得很冗余，收到郭嫂在litepal 新版本的启发，一个库应该纯粹，适当的时候可以做减法，将不必须要的东西都是删除，所以才有了今天的这个框架，BaseMVVM
